✨ KEY CSS TECHNIQUES USED:

1. CSS clamp()
   Example: font-size: clamp(1.5rem, 5vw, 5rem)
   → Scales between 1.5rem and 5rem based on viewport width

2. Flexbox
   → Responsive navigation & buttons

3. CSS Grid
   → Responsive multi-column layouts
   → Auto-adjust columns with grid-template-columns

4. Media Queries
   → Specific rules for each screen size

5. Viewport Meta Tag
   <meta name="viewport" content="width=device-width, initial-scale=1.0" />
